LLM-to-LLM Communications Vocabulary

Welcome to the LLM Communications Vocabulary Definitions

This repository contains a framework that helps developers structure communication between large language models (LLMs) and code. It offers a set of vocabulary definitions designed to facilitate LLM-to-LLM interactions using JSON-LD format. By providing a clear structure for defining the types of data being sent and received, this project aims to make the communication between systems and LLMs more transparent and effective.

The goal is to enable developers to easily define data contexts that allow LLMs and other systems to understand what type of information is being exchanged, improving clarity and standardisation in the communication process.
